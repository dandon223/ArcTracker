Main menu
=========
.. spec:: New game
    :id: SPEC_MAIN_MENU_1
    :status: open

    ArcTracker shall have `new game` button that allows user to start a new game.

.. spec:: Games
    :id: SPEC_MAIN_MENU_2
    :status: open

    ArcTracker shall have `games` button that allows user to view previous games.

.. spec:: New player
    :id: SPEC_MAIN_MENU_3
    :status: open

    ArcTracker shall allow user to add new player.

.. spec:: Players
    :id: SPEC_MAIN_MENU_4
    :status: open

    ArcTracker shall allow user to view added players.
